# 問題の追加方法（学科15〜26などを追加するとき）

問題データはすべて `data.js` に入っています。テキストエディタ（メモ帳等）で開き、
配列の最後（`];` の直前）に、次の形式で追記してください。

```javascript
 {
  "id": "15-1",
  "cat": "学科15",
  "q": "問題文（もんだいぶん）をここに書（か）きます。",
  "a": true,
  "exp": "解説（かいせつ）をここに書（か）きます。（99P）",
  "img": null,
  "vi": {"q": "ベトナム語の問題文", "e": "ベトナム語の解説"},
  "ne": {"q": "ネパール語の問題文", "e": "ネパール語の解説"},
  "en": {"q": "英語の問題文", "e": "英語の解説"},
  "idn": {"q": "インドネシア語の問題文", "e": "インドネシア語の解説"}
 },
```

## 各項目の意味

| 項目 | 内容 |
|---|---|
| id | 「学科番号-問題番号」。重複しないこと |
| cat | 学科名。新しい名前を書けばスタート画面にボタンが自動で増えます |
| q | 問題文。ふりがなは「漢字（かんじ）」の形式で書くと、アプリが自動でルビ表示に変換します |
| a | 〇が正解なら `true`、✖が正解なら `false` |
| exp | 解説文（間違えたときに表示） |
| img | 標識などの画像がある場合はファイル名（例 `"g15_q1.png"`）。画像は `images` フォルダに入れる。なければ `null` |
| vi / ne / en / idn | ベトナム語・ネパール語・英語・インドネシア語訳。未訳の間は日本語をそのまま入れてもアプリは動きます（idnはインドネシア語。idは問題番号に使用済みのため） |

## 注意

- 記号（カンマ、引用符 `"`）の消し忘れ・付け忘れがあるとアプリが動かなくなります。
  1問追加するごとにブラウザで開いて動作確認するのが安全です
- Wordファイルのまま追加したい場合は、ファイルをClaudeに渡して
  「この学科ファイルをdata.jsの形式に変換して」と頼めば同じ形式で変換できます
